# YouTour главная страница

Одностраничный проект, выполненный в рамках тестового задания на вёрстку.
Проект представляет собой адаптивную страницу с использованием модульной структуры SCSS и сборки через Webpack.

## Структура проекта

```
youtour/
├── src/
│   ├── fonts/              # Подключаемые шрифты
│   ├── images/             # Изображения проекта
│   ├── styles/
│   │   ├── base/           # Базовые стили: reset, fonts
│   │   ├── blocks/         # SCSS-блоки страницы
│   │   ├── utils/          # Переменные и миксины
│   │   └── index.scss      # Главный файл стилей
│   ├── index.html          # Основной HTML
│   └── index.js            # Точка входа JS
│
├── package.json
├── package-lock.json
├── webpack.config.js       # Конфигурация сборки
└── README.md
```

### Важные файлы

- `src/index.html` — свёрстанная страница
- `src/styles/index.scss` — собирает всё SCSS
- `src/styles/utils/_variables.scss` — палитра и переменные
- `webpack.config.js` — настроенная сборка, обработка SCSS/JS/asset-файлов

## Что было сделано

- Верстка макета страницы YouTour
- Настройка адаптивности
- Подключение шрифта (Roboto)
- Структурирование SCSS по методологии компонентного подхода
- Настройка Webpack для сборки (SCSS → CSS, минификация, копирование ассетов)
- Импорт изображений и подготовка структуры проекта

## Реализованный функционал

- Собранная адаптивная HTML-страница
- Корректная организация файловой структуры
- Обработка стилей SCSS через Webpack
- Поддержка переменных, миксинов и отдельных блоков
- Импорт изображений через Webpack
- Автоматическая сборка и запуск дев-сервера

## Используемые технологии

- `HTML5` / `CSS3` / `SCSS`
- `JavaScript (ES6+)`
- `Webpack 5`
- `File-loader` / `Asset modules`
- `Node.js` / `npm`

## Требования к запуску

- `Node.js` **16+**
- `npm` **8+**

### Инструкция

1. Клонировать репозиторий:
   `git clone https://github.com/MilanaMur/you-tour-homepage.git`
2. Перейти в папку проекта:
   `cd you-tour-homepage`
3. Установить зависимости:
   `npm install`
4. Запустить дев-сервер:
   `npm run start`
